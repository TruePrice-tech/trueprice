Santa Ana electrical FAQ

Can I install recessed lighting myself?

While physically possible, running new circuits and making connections in existing walls carries fire and shock risk. Most jurisdictions require a licensed electrician and a permit for new fixture circuits.

How much does it cost to add an outlet?

Adding a standard outlet costs $150-$300 if wiring can be run through accessible spaces. If walls need to be opened or a new circuit is required, expect $300-$600 per outlet.
