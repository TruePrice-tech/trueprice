# Backlink Outreach Email Templates

These are link-acquisition pitches (asking someone to ADD a link to YOUR site in their existing content). The widget-embed templates in outreach-emails.md are a separate, complementary play.

Personalize every email. Generic templates get filtered to spam. Always reference a SPECIFIC article you read.

---

## Template 1: Resource Page Add (Highest Hit Rate)

Find pages titled "best home cost calculators", "useful homeowner resources", "links for patients", etc. Then ask to be added.

Search operators that find these:
- `inurl:resources "home improvement"`
- `inurl:links "cost calculator"`
- `intitle:"resources" "home services"`

**Subject:** Quick suggestion for your [page topic] resource list

Hi [Name],

I was reading your [exact page title] and noticed you link to [specific resource on their list, e.g. HomeAdvisor's True Cost Guide]. Useful list.

I run TruePrice (https://truepricehq.com), a free homeowner tool that breaks down local contractor pricing by city, material, and contractor tier. Unlike most cost-guide sites, we show ranges and let visitors paste their actual quote for a free analysis -- no email required, no lead-gen sales calls.

Might be a fit for that resource list if you ever update it. Specifically, this page has the same format as the others you link to: https://truepricehq.com/guides.html

Either way, thanks for keeping that page maintained. It's a genuinely useful list.

Lane Adams
TruePrice

## Template 3: Broken Link Replacement

Find a broken outbound link on their site, point it out, suggest yours as the replacement.

Tools to find broken links: Ahrefs Broken Link Checker (free trial), or Chrome ext "Check My Links".

**Subject:** Broken link on [page title]

Hi [Name],

Quick heads up -- the link to [old domain] in your [page title] (in the section about [topic]) returns a 404 now. Looks like that site shut down in [year].

If you're updating, we have a similar resource you could link instead: https://truepricehq.com/[specific-guide].html. It covers [the same topic with brief specifics].

## Quick Tips

**Send rate:** 5-10 personalized emails/day max. Mass-send and you torch your sender reputation.

**Follow-up:** ONE follow-up after 7 days. "Just bumping this in case it got buried." Then drop it.

**Subject lines:** Lowercase, conversational, no clickbait. "quick question" outperforms "AMAZING resource for you!!!".

**Sign-off:** Use a real name + signature. No corporate template. People link to people, not brands.

**Track:** Keep a CSV of who/when/response. Avoid double-emailing the same domain.

**Use what you already have:** outreach-targets.json has 22 vetted targets. Run these scripts against that list first.
